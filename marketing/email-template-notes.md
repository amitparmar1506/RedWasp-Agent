# Restaurant AI Agent Email Template Notes

Use `restaurant-ai-agent-email-template.html` for outreach to restaurants.

Replace these placeholders before sending:

- `https://YOUR-DEMO-LINK.com`
- `https://YOUR-DEMO-VIDEO-LINK.com`
- `https://YOUR-CALENDAR-LINK.com`
- `https://YOUR-DOMAIN.com/images/redwasp-ai-hero.jpg`
- `https://YOUR-DOMAIN.com/images/redwasp-ai-video-thumbnail.jpg`
- `https://YOUR-DOMAIN.com/images/chatbot-screenshot-1.jpg`
- `https://YOUR-DOMAIN.com/images/chatbot-screenshot-2.jpg`
- `[Owner/Manager Name]`
- `[Restaurant Name]`
- `[Your Name]`
- `[Company Name / MITEX AI]`
- `[Phone Number]`
- `[Website]`

Email clients usually block embedded video playback, so the template uses a clickable video thumbnail. Host the video on a landing page, YouTube, Vimeo, Google Drive, or your own website and link the thumbnail to it.
